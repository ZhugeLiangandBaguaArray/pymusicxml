valign data type

The valign type is used to indicate vertical alignment to the top, middle, bottom, or baseline of the text. Defaults are implementation-dependent.

Base type: token

Allowed values

top Aligned to the top of the text.
middle Aligned to the middle of the text.
bottom Aligned to the bottom of the text.
baseline Aligned to the baseline of the text. If the text is on multiple lines, algned to the baseline of the lowest line of text.
