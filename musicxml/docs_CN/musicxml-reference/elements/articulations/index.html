The <articulations> element

Parent element: <notations>


The <articulations> element groups together articulations and accents.

Content

Zero or more of the following

  • <accent>
  • <strong-accent>
  • <staccato>
  • <tenuto>
  • <detached-legato>
  • <staccatissimo>
  • <spiccato>
  • <scoop>
  • <plop>
  • <doit>
  • <falloff>
  • <breath-mark>
  • <caesura>
  • <stress>
  • <unstress>
  • <soft-accent>
  • <other-articulation>

Attributes

Name Type Required? Description
id ID No Specifies an ID that is unique to the entire document.

Examples

This element is used in the following examples:

<accent>, <articulations>, <breath-mark>, <caesura>, <chord> (Multiple Stop), <detached-legato>, <doit>, <falloff>, <plop>, <scoop>, <soft-accent>, <spiccato>, <staccatissimo>, <staccato>, <stress>, <strong-accent>, <tenuto>, <unstress>
