The <beat-type> element

Parent elements: <interchangeable>, <time>


The <beat-type> element indicates the beat unit, as found in the denominator of a time signature.

Content

string

Attributes

None.

Examples

This element is used in the following examples:

<assess> and <player>, <attributes>, <barline>, <beat-type>, <beats>, <concert-score> and <for-part>, <double>, <instrument-change>, <interchangeable>, <part-symbol>, <pedal> (Lines), <score-timewise>, <staff-type>, <staves>, <unpitched>, Tutorial: Après un rêve, Tutorial: Chopin Prelude, Tutorial: Chord Symbols, Tutorial: Hello, World, Tutorial: Percussion, Tutorial: Tablature
