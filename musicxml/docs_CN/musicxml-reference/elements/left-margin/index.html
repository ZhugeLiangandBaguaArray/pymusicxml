The <left-margin> element

Parent elements: <page-margins>, <system-margins>

The <left-margin> element specifies the left margin for the parent element in tenths.

Content

tenths

Attributes

None.

Examples

This element is used in the following examples:

<attributes>, <defaults>, <system-distance>, <system-dividers>, Tutorial: Après un rêve, Tutorial: Chopin Prelude, Tutorial: Chord Symbols, Tutorial: Percussion, Tutorial: Tablature
